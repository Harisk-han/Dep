# Dynamic Blog Website

## Project Description

This project is a dynamic blog website built using Node.js, Express, and MongoDB. It allows users to create, read, update, and delete blog posts. The project demonstrates the implementation of a full CRUD (Create, Read, Update, Delete) application with a focus on backend development using Node.js and Express, and data storage using MongoDB.

## Features

- Create new blog posts
- View all blog posts
- Update existing blog posts
- Delete blog posts
